# BNR-PE Documentation Hub

This documentation set is intended to support research progression with a reproducible, inspectable implementation and clear operating signals.

## What This Covers

- Motivation and scope of BNR-PE as a positional encoding direction.
- Theory-to-code mapping for the JAX reference implementation.
- Pedagogy and executable learning assets.
- Auto-generated API reference from package source.
- Auto-generated context snapshot from benchmark/gate artifacts.
- Governance and process docs used for go/no-go decisions.

## Quick Start

```bash
cd bnrpe_jax
python -m pip install -r requirements.txt -r requirements-docs.txt
python scripts/build_docs.py --repo-root ..
```

Then view the static site:

```bash
python -m mkdocs serve --config-file ../mkdocs.yml
```

## Auto-Updating Contract

API and context pages are generated by scripts:

- `python scripts/generate_api_docs.py`
- `python scripts/generate_context_docs.py`

`python scripts/build_docs.py --repo-root ..` runs both generators before building the site.
